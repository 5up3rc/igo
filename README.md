igo
===

A simple interactive Go interpreter built on [go-eval](github.com/sbinet/go-eval) with some readline-like refinements


## Example

```sh
$ igo
igo> func f() { println("hello world") }
igo> f()
hello world
```

## Documentation

http://godoc.org/github.com/sbinet/igo
